# 🏛️ Poke App Project

Aplicación Front-End construida con **React**, **Vite** y **Tailwind CSS v4.3** orientada a la gestión y consumo de datos de la API oficial de Pokémon.

## 🛠️ Tecnologías Utilizadas

* **React** (Biblioteca para la interfaz de usuario)

* **Vite** (Herramienta de empaquetado ultra rápida)

* **Tailwind CSS v4** (Ecosistema de estilos moderno)

* **pnpm** (Gestor de dependencias optimizado)
